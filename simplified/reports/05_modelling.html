Evaluation Fundamentals ¶

To evaluate a classification model the five most common metrics are:

  • Accuracy
  • Precision
  • Recall
  • F1-Score
  • AUC (Area under the ROC curve)

Since the dataset is balanced in regards to the target variable and there is no tangible distinction between type I and II errors, there is no need to look too much into Precision, Recall and consequently F1-score. The primary goals are to predict accurately the winner of the match and output a winning probability as close as possible to reality. Therefore, the two most important metrics are: Accuracy and ROC-AUC .

In any case, it must be ensured that the model is not considerably biased to predict one of the classes even with balanced team features values.

Store fitted models ¶

In [35]: 
best_estimators = []
for name, gs in hyp_searches.items():
    if name in ('Linear SVC', 'Ridge'):
        # CalibratedClassifierCV acts like a wrapper to enable the usage of the predict_proba method
        best_estimator = CalibratedClassifierCV(gs.best_estimator_)
    else:
        best_estimator = gs.best_estimator_
    best_estimator.fit(X_train, y_train)
    # get estimator class name and remove parentheses (From LinearSVC() to LinearSVC)
    estimator_name = name.replace(" ", "_")

    current_date_string = datetime.utcnow().strftime("%Y_%m_%d_%H_%M_%S")
    model_folder_path = MODEL_DIR_S / f'model_{estimator_name}_{current_date_string}'

    # create models folder
    os.mkdir(model_folder_path)
    
    # save models
    joblib.dump(best_estimator, f'{str(model_folder_path)}/model.joblib')
    best_estimators.append((name, best_estimator))
